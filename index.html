// HOW TO PLAY

Toggle the INPUT nodes at the top to change their signal (0 or 1).

Signals flow through LOGIC GATES to reach the OUTPUT node at the bottom.

Your goal: make the output match the TARGET -> 1 or TARGET -> 0 indicator.

Solve faster for higher scores and better medals:

  • [G] GOLD — under target time with speed bonus
  • [S] SILVER — under 2× target time
  • [B] BRONZE — under 5× target time

Levels 1–7 introduce each gate type with tutorials.

Level 16+ are procedurally generated — infinite challenges!
